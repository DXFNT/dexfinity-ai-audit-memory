NIKDY nedávať do auditov čísla vycucané z prsta. Sme experti — každý výstup je o našu povesť.

**Why:** User explicitly said: "nic čo je vycucané z prsta si nemôžeme dovoliť dávať do našich auditov... nezabudaj že sme experti pri každom jednom výstupe ide o našu povesť." Incident: "100K+ do 12 mesiacov" a "10-20% ležiakov za 4-5 mesiacov" boli v audite bez akejkoľvek metodológie.

**How to apply:**
- Every number must have a source (Ahrefs, Google SERP, Ad Library, Heureka...)
- If you can't back a claim with data → don't include it
- If unsure → ask the user before publishing
- Verify SERP positions in live Google, don't trust Ahrefs blindly
- "Odhad" or "potenciál" sections must clearly state methodology or be removed
- Also applies to: email addresses (use hello@dexfinity.com not hello@dex.sk), logo URLs, any factual claims
- Better to have fewer claims that are bulletproof than many that are questionable
