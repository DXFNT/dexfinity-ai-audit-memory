Prečo práve tento sled

Poradie je funnel. Hodnota najskôr (sekcie 3–16, buduje dôveru), FOMO v strede (17 — 3× locked), reframe ceny na konci (18 — wasted spend box pred pricing). Wasted spend box musí byť pred pricing kartami, inak funnel nefunguje.

3 · Signature komponenty

Niektoré komponenty sa na audite „chytili" — identifikujú Dexfinity audit na prvý pohľad. Sú patentované v CSS library.

Flip cards na platbu

Pricing karta sa otočí na IBAN údaje pre bankový prevod — bez modálu, bez odchodu z kontextu.

  • .flip-front = position relative (určuje výšku)
  • .flip-back = position absolute (overlay)
  • Hover transform MUSÍ byť vypnutý v flip kontexte
  • Copy-to-clipboard na IBAN (small inline JS)

Locked sekcie s unlock CTA

3 sekcie s filter: blur(6px), overlay s lock ikonou, scroll-to-pricing CTA.

  • ROAS kalkulácia, budget leaks, akčný plán
  • Navy 64px circular lock icon, white SVG
  • Badge „Pro klientov" + „Odomknúť →"
  • Smooth scroll do #pricing

Finding bars (severity)

4px farebný bar vľavo = priorita findingu. Žiadne semaphore kruhy ani emoji.

  • .bar.high = navy (kritické)
  • .bar.med = blue (stredné)
  • .bar.low = blue 30% (nice-to-have)
  • Odporúčanie v .rec boxe pod finding textom

Wasted spend context box

Nad pricing kartami 3-stĺpcový grid: spend / wasted % / ROI payback. Klient číta „stratím viac ako zaplatím za audit."

  • Navy-transparent bg nad pricing
  • Biely spend / orange wasted / teal payback
  • Povinná komponenta — bez nej funnel stráca silu

4 · Case studies — progression z 3 týždňov

Každý audit niečo nové priniesol. Tieto tri sú míľniky evolúcie.

Týždeň 1 · Baseline

Aquapond

Prvý gold-standard audit. Zaviedli sme 18-sekciovú architektúru, flip cards, locked sekcie, wasted spend box.

  • Definovaná paleta a typografia
  • Flip card platobný UX
  • Sezónny SVG chart s „STE TU" pulzom
  • Pricing: 1 290 € + 1 690 € setup
Týždeň 2 · Evolúcia

Lamely3D (Deep)

Multi-trh variant. Priniesol executive summary floating box, quick wins framing, horizontálne bar charty, multi-trh dashboard SK/CZ/DE/AT.

  • Executive summary floating pod hero
  • Color-coded highlight boxes (positive/warning)
  • Ad creative showcase s Library ID
  • Print styles pre PDF export
Týždeň 3 · Dospievanie

Oxalis Tea

Double-check rule sa narodila tu — agent deklaroval „Heureka SK neexistuje" na základe 404. Realita: profil bol pod iným slug-om.

  • Zavedené overovanie 2× pre negatívne findings
  • „Na overenie" badge namiesto „Kritické"
  • Gap analysis standard sa ukotvilo
  • Logo min sizes (48/64 px) po „Go Beyond" incidente

5 · Pravidlá & anti-patterns

Pravidlo obsiahnuté v memory súbore = tím ho už nemusí hľadať. Týchto 11 feedback súborov pokrýva najčastejšie chyby.

Anti-pattern
Prekrývajúce sa CTA tlačidlá

Primary + secondary button bez dostatočného paddingu, na mobile sa tap tapne zle.

Správne
Action bar pattern

Flex row s gap: 12, alebo oddelený kontajner. Po implementácii VŽDY screenshot.

Anti-pattern
Logo pod 48 px v nave

„Go Beyond" tagline sa stratí, brand feel zmizne.

Správne
Nav 48, Footer 64, Hero 80

Footer je brand anchor — musí byť výrazný.

Anti-pattern
Hover transform v flip karte

translateY(-4px) + 3D rotate = rozbitá perspektíva.

Správne
Vypnuté hover v flip kontexte

.flip-card .pricing-card:hover { transform: none; }

Anti-pattern
Plain text Gmail drafty

Signature sa nerenderuje, rozbité formátovanie, žiadna branded hlavička.

Správne
Vždy contentType: html + signature

API signature neautomaticky nepridá — pripojiť manuálne.

Anti-pattern
Neoverené negatívne findings

„X neexistuje" na základe 404 → klient vyvráti → credibility kaput.

Správne
Double-check rule

Variácie názvov, site: search, cross-reference webu klienta. Ak nejde overiť → „Na overenie" badge.

Anti-pattern
Generic farby (#00D4AA teal)

Jeden hex mimo palety = vizuál vyzerá ako SaaS template.

Správne
CSS variables only

Nikdy inline hex — vždy var(--dex-teal).

Double-check rule — najcennejšie learning

Každý finding tvaru „X neexistuje" / „0 Y" / „404" / „Neviditeľný" MUSÍ byť overený dvakrát nezávisle pred zaradením do auditu. Variácie názvov (s/bez pomlčky, s/bez suffix), Google site: search, cross-reference na webe klienta. Ak nejde overiť (403, block) → badge „Na overenie", nie „Kritické". Jedna vyvrátená tvrdá veta spochybní celý audit.
